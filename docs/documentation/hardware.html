Hardware

What you'll need:

  • Raspberry Pi 3B/B+
  • USB Microphone or USB Webcam with embedded microphone
  • USB speakers
  • 8GB SD Card (make sure it's flashed)
  • USB Wall Charging Adapter
  • HDMI cable
  • Raspberry Pi case (optional but recommended)
  • Monitor for setup
  • Wireless internet connection (if not, use an ethernet cable)


    • Assembly

    Connect the SD card, microphone, micro-USB cable, and speakers into the Raspberry Pi. For now, you're going to need to connect the Pi to a monitor so as to configure everything, so make sure to do that as well. If you have a case, assemble it and put it on the Pi.
