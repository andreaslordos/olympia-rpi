Configuration

You'll need to follow this step to get all your API keys, which are needed for services such as streaming, getting the weather, etc. All API keys for all current and future apps that you may build should be stored in the file api_keys.txt, which is found in the resources folder.



Wolfram Alpha API Key

1) To allow Olympia to answer any question using the Wolfram Alpha API, you must acquire a Wolfram Alpha API key. Visit this site and sign up as a developer

2) Click "Get an AppID after signing in. Enter any application name and description

3) Add your API key on the api_keys.txt file, on the line that starts with questions.py



Note: It's important when building other apps to follow this same convention. If your module uses an API key, make sure that the module reads api_keys.txt folder using the "read_api_key" function. Store API keys in the format of [name of module.py]~[API key]. For example, "question.py~121334234593284023"
