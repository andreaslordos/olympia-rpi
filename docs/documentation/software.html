Software

Use method #1 if you have a way to modify the contents of an SD card (e.g. your laptop has an SD card reader)



Method #1:

1) Download this image that contains everything ready-to-go

2) Extract the contents of the zip file

3) Follow these instructions to load up the image on an SD card

4) All done!



Method #2:

1) Download NOOBS Lite

2) Follow this tutorial to load up your SD card with NOOBS

3) Insert the SD card in your Raspberry Pi. Power on the Pi and install the Raspbian Jessie OS

4) Login with the username "pi" and the password "raspberry"

5) Type 'sudo-raspi-config'

6) Change the password of the Raspberry Pi to anything other than 'raspberry', enable boot to console autologin (option 3 --> option 2), set time to local time

7) Type 'sudo reboot now'

8) After the reboot, type 'sudo apt-get update' and 'sudo apt-get upgrade'

9) 'sudo apt-get install git'

10) 'git clone http://github.com/andreaslordos/olympia-rpi'

11) 'wget https://bootstrap.pypa.io/get-pip.py'

12) 'sudo python get-pip.py'

13) 'sudo apt-get install python3-dev'

14) 'sudo python3 get-pip.py'

15) 'sudo apt-get install' the following: python-dev, libxml2-dev, libxslt1-dev, portaudio19-dev, mplayer, flac, libatlas-base-dev, vlc

16) 'sudo pip2 install pyaudio'

17) 'cd olympia-rpi'

18) 'sudo pip3 install -r requirements.txt'

19) Done!
