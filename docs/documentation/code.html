Code your own modules

Learn how to code your own modules by building one



Introduction

An important thing to understand is that each module is essentially a function, that takes an input, does something with it and then returns what Olympia should say back to the user.

Take for example the News Update module. As an input, it takes in what the user said, uses lines 17-28 to determine which news website the user specified (if any) and then parses the RSS feed of that source. It then returns newsTuple, which contain the title of the first three entries of the RSS feed'
